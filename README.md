# StayHealthy - Medical Appointment Booking

A frontend capstone project for booking medical appointments online with verified doctors.

## Project Name
**StayHealthy** - Online Medical Appointment Booking Platform

## Features
- Search for doctors by specialty
- View availability and book appointments
- Instant consultation booking
- User authentication (Sign Up / Login)
- Patient profile management
- Appointment notifications
- Patient feedback and reviews
- Medical reports view and download
- Responsive design for mobile and desktop

## Technologies Used
- HTML5, CSS3, JavaScript
- React.js (Vite)
- React Router DOM
- Node.js / Express.js
- MongoDB (Atlas)
- JWT Authentication

## Setup Instructions

### Prerequisites
- Node.js installed
- MongoDB Atlas account

### Installation

1. Clone the repository:
git clone https://github.com/dsbly1/med_appt.git

2. Navigate to the project folder:
cd med_appt

3. Install dependencies:
npm install

4. Navigate to server folder and install server dependencies:
cd server && npm install

5. Update MongoDB connection string in server/db.js

6. Start the server:
node index

7. In a new terminal, run the client:
npm run preview

8. Open browser at http://localhost:4173
